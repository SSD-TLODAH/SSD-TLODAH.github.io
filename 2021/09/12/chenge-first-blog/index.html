chenge-first-blog

（很抱歉用了投机取巧的方法，现在修改重发）
1.工具准备
注册github，并建立仓库（右上角的“+”点开，点击New repository，仓库起名为：username.github.io，并勾选“add a README file”）
安装git（http://git-scm.com/）
安装node.js（https://nodejs.org/en/download/）
安装hexo及其依赖（在git bash here中输入npm install hexo -g及npm install – save hexo-deployer-git）
2.配置SSH
打开git bash here
输入git config –global user.name“github用户名”
输入git config –global user.name“github注册邮箱”
输入ssh-keygen -t rsa -C“github注册邮箱”
按三次回车
在C:\users\username.ssh中找到id_rsa.pub文件，复制
打开github的settings，在SSH and GPGkeys 中点击add ssh key，将复制内容粘贴至key（title随意）
3.搭建博客
新建文件夹，在文件夹中使用git bash here
输入npm install hexo –g hexo-cli
输入hexo init
输入hexo generate
输入hexo server
在浏览器中输入http://localhost:4000/
此时有博客出现
4.将博客部署在github上
打开博客根目录的文件_config
将其末尾改为
deploy：
type:git
repository:git@github.com:username/username.github.io.git
branch:main
在根目录打开git bash here
输入hexo d
5.编写博客
在根目录打开git bash here
输入hexo new ‘chenge-first-blog’
在博客文件的source_posts中打开chenge-first-blog.md
在末尾编写内容
